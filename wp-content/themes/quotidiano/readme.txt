== Description ==

Quotidiano is a clean WordPress theme that can be used to create a magazine or newspaper websites. Fully customizable and optimized for the mobile devices, Quotidiano provides a grid of recent articles and an optional header banner, is also compatible with WooCommerce, Contact Form and Elementor plugins.

Developed by ThemeinProgress, https://www.themeinprogress.com

Quotidiano WordPress Theme is a child theme of Avventura Lite Theme, Copyright 2022 ThemeinProgress.
